# Air Quality Monitoring System

## Overview
This project is an ESP32-based Air Quality Monitoring System that monitors air quality using the MQ135 gas sensor. The system displays air quality information on an I2C LCD and alerts users through LEDs and a buzzer when pollution levels become unhealthy.

## Features
- Real-time air quality monitoring
- AQI indication
- LED status indication
- Buzzer warning
- LCD display
- ESP32-based implementation

## Components Used
- ESP32 Development Board
- MQ135 Gas Sensor
- I2C LCD Display (16x2)
- Buzzer
- LEDs
- Jumper Wires
- Breadboard

## Software Used
- Arduino IDE
- C++

## Future Improvements
- Wi-Fi monitoring
- Mobile app integration
- Cloud data storage
- Historical data visualization
